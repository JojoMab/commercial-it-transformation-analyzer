# Recruiter Summary

## Project In One Sentence

The Commercial IT Transformation Analyzer is a Python project that evaluates realistic customer, project, workshop and service data and turns it into prioritized recommendations for digitization and consulting projects.

## Why It Fits Business Information Systems

- connects computer science and business administration in one visible project
- uses data modeling, CSV processing and SQLite
- evaluates projects by value, margin, effort, risk and payback
- shows consulting-oriented thinking through workshop and readiness factors
- produces concrete reports that a business department could use

## Fit For Bosch

The project connects to Commercial IT, ERP/database topics, Power BI-style transparency, process digitization and service KPIs in a Building Technologies context.

## Fit For Fujitsu

The project connects to Sales & Consulting, customer workshops, proposal preparation, modernization initiatives and economic evaluation of IT projects.

## Technical Impression

- runs directly with `python3 main.py`
- no external dependencies
- reproducible sample data
- includes a unit test
- includes a GitHub Actions workflow
- clear README and documentation
